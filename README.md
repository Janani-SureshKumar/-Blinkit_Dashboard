# Blinkit Analytics Dashboard – Power BI Report  

This repository contains a comprehensive Power BI report built from scratch that analyzes Blinkit’s operations and customer analytics to uncover critical trends in order volumes, product-category demands, delivery performance, and key business KPIs across multiple regions.  
This project showcases how data transform & visualization can empower operational decision-making and strategic planning.

---

## Key Features  
• Analyze order volumes, delivery times, and product category performance.  
• Identify peak shopping hours, high-demand zones, and segment customer behavior.  
• Understand delivery-fleet efficiency, payment method trends and region-wise performance.  
• Track revenue growth, average order value and operational bottlenecks.

---

## Tools Used  
• Power BI Desktop – Data modelling, DAX measure creation & report design.  
• Power Query – Extract, clean and transform raw Blinkit-style data.  
• GitHub – Version control & portfolio showcase.  
• Excel / CSV – Raw dataset preparation.

---

## Data Model  
The report is based on two main tables:  
• **Orders Table:** Contains details such as Order ID, Order Time, Delivery Time, Product Category, Order Value, Payment Type and Region.  
• **Location/Customer Table:** Maps location/region codes to full names, customer segments, and supports region-wise or zone-wise aggregation.

## Insights Generated  
• Highest order volume occurs between 12 PM-6 PM, peaking around 3 PM.  
• Most frequent product category: Groceries, accounting for ~35% of orders.  
• Delivery performance: Average delivery time improved by ~18% year-on-year in urban zones.  
• Payment trend: Digital wallet payments account for ~62% of transactions, cash ~38%.  
• Region-wise snapshot: Urban Central zone recorded highest average order value (₹1,250) and fastest delivery time (22 min).  
• Demand zones: Suburban West and South saw demand spikes on weekends (Saturday & Sunday >20 k orders).  
• Product demand shift: Plant-based and health-snack categories grew by ~40% in the last 6 months.
